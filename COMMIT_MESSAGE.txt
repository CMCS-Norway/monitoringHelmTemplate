feat: initial release of monitoring-stack Helm chart v1.0.0

Add production-ready Helm chart for comprehensive monitoring infrastructure
with Azure, network, and HTTP probing exporters.

## Features

### Exporters (4)
- Azure KeyVault Exporter (v1.0.12) - Key Vault and certificate monitoring
- Azure Resource Manager Exporter (v1.3.5) - Azure resource costs, health, security
- Fortigate Exporter (v3.0.0) - Network firewall monitoring (25+ locations)
- Blackbox Exporter (v11.3.1) - HTTP/HTTPS/TCP/ICMP endpoint probing

### Architecture
- Hybrid chart structure (3 Helm dependencies + 1 custom deployment)
- Unified configuration via single values.yaml
- External Secrets integration for credential management
- Prometheus ScrapeConfigs for all exporters
- Conditional resource deployment (enable/disable per exporter)

### Resources
- 12 Kubernetes resource templates
- 4 documentation files (README, DEPLOYMENT, HELM_VALUES, VERIFICATION)
- GitHub Actions workflow for CI/CD
- Complete values reference with 515+ configuration options

### Security
- No hardcoded secrets (External Secrets Operator integration)
- Pod security contexts enforced
- Read-only root filesystems
- Non-root containers

### Documentation
- Quick start guide
- Step-by-step deployment procedures
- Complete values reference
- Pre/post-deployment validation guides
- Troubleshooting documentation
